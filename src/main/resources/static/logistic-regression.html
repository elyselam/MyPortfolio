Will a Customer Click on the Ad? A Logistic Viewpoint¶

Using several variables from an advertising dataset, I build a logistic regression model to predict whether a customer will click on an ad.

Plain-English Analysis of each cell's code is coming soon.

This data set features the following variables:

  • 'Daily Time Spent on Site': consumer time on site in minutes
  • 'Age': cutomer age in years
  • 'Area Income': Avg. Income of geographical area of consumer
  • 'Daily Internet Usage': Avg. minutes a day consumer is on the internet
  • 'Ad Topic Line': Headline of the advertisement
  • 'City': City of consumer
  • 'Male': Whether or not consumer was male
  • 'Country': Country of consumer
  • 'Timestamp': Time at which consumer clicked on Ad or closed window
  • 'Clicked on Ad': 0 or 1 indicated clicking on Ad

Logistic Regression¶

Now it's time to do a train test split, and train our model!

You'll have the freedom here to choose columns that you want to train on!

Split the data into training set and testing set using train_test_split

In [14]:
from sklearn.model_selection import train_test_split
In [32]:
X=ad_data[['Daily Time Spent on Site', 'Age', 'Area Income','Daily Internet Usage', 'Male']]
y= ad_data['Clicked on Ad']
X_train, X_test, Y_train, Y_test = train_test_split(X,y,test_size=.3, random_state=101)

Train and fit a logistic regression model on the training set.
